## Quick Start

### 1. Install

```bash
npm install -g deepseek-coder-cli
```

### 2. Set API Key

Get your API key from: https://platform.deepseek.com/api_keys

```bash
deepseek --key YOUR_API_KEY
```

This saves your key to `~/.agi/secrets.json`.

Or set via environment variable:
```bash
export DEEPSEEK_API_KEY=your_key_here
```

```bash
# Interactive mode
deepseek

# With initial prompt
deepseek "explain this codebase"

# Quick mode (non-interactive)
deepseek -q "list all TypeScript files"
```

### Lean Agent (Alternative)

A streamlined coding agent with unified tools:

```bash
# Run lean agent

# Or after global install
```
